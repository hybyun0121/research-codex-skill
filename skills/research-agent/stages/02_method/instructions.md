# Method

## Goal

Turn the selected research direction into a rigorous method specification.

## Workflow

1. Restate the method claim.
2. Define inputs, outputs, variables, assumptions, objective, and constraints.
3. Write mathematical formulation where possible.
4. Check rigor:
   - undefined symbols
   - dimensional inconsistency
   - hidden assumptions
   - invalid optimization objective
   - data leakage
   - mismatch between novelty claim and formulation
5. Produce pseudocode for algorithmic steps.
6. Add a visual explanation or diagram plan when helpful.
7. Compare against the closest prior methods.

## Output

Write `research/method.md` and update:

- `method.claim`
- `method.formulation`
- `method.algorithm`
- `method.visuals`
- `method.rigor_review`
- `method.comparison_to_prior_work`
